Loading

This demo allows users to show a loading image while Embedly fetches the data and then display the content.

demo

Video

code

  <div class="video">
    <div class="loading"><img src="img/loading.gif"/></div>
    <a href="http://vimeo.com/18150336">Video</a>
  </div>
  <script>
  $(document).ready(function() {
    $('.loading').show();
    $(".video a").embedly({maxWidth: 500,
                       key : 'Your Embedly Key',
                       method : 'replace'}).bind('embedly-oembed', function(e){
                         $('.loading').hide();
                       });
  });
  >/script>
